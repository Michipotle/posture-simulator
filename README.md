# 🏥 Simulador de Higiene Postural - Prevención de Lesiones Musculoesqueléticas

Un juego interactivo educativo para enseñar a trabajadores cómo proteger su columna vertebral en actividades cotidianas del trabajo y el hogar.

## 🎮 Descripción

Este simulador proporciona un aprendizaje gamificado sobre ergonomía y higiene postural. Los usuarios deben tomar decisiones correctas en 7 escenarios diferentes para proteger su "salud vertebral" (medida en puntos del 0-100%).

**Características principales:**
- ✅ 7 niveles interactivos con situaciones reales
- ✅ Sistema dinámico de salud con barra de progreso visual
- ✅ Feedback educativo con información médica
- ✅ Puntuación final con recomendaciones personalizadas
- ✅ Interfaz responsiva (móvil, tablet, desktop)
- ✅ Diseño moderno y atractivo con emojis y animaciones

## 📋 Niveles del Juego

| Nivel | Escenario | Contexto |
|-------|-----------|----------|
| 1 | 💻 Trabajando en la Oficina | Postura correcta frente a computadora |
| 2 | 📦 Levantando Cajas Pesadas | Técnica segura de levantamiento |
| 3 | 🛒 Cargar Bolsas de Compra | Distribución equilibrada de peso |
| 4 | 🧹 Limpiando el Hogar | Postura durante tareas domésticas |
| 5 | 🪜 Trabajos en Altura | Seguridad en escalera/taburete |
| 6 | 🚗 Conduciendo Vehículo | Ergonomía en auto |
| 7 | 🏆 Evaluación Final | Consolidación de aprendizajes |

## 🎯 Contenido Educativo

Cada nivel incluye:
- **Pregunta con 3 opciones** (correcta + 2 incorrectas)
- **Feedback inmediato** con términos médicos:
  - Protrusión cervical
  - Hernia discal
  - Radiculopatía
  - Cervicalgia
  - Lumbalgia
  - Y más...
- **Consejos profesionales** después de respuestas incorrectas
- **Sistema de daño:** Las malas posturas restan puntos de salud

## 💡 Consejos Ergonómicos Clave

El simulador enseña estas prácticas:

### En la Oficina
- ✓ Monitor a altura de ojos
- ✓ Codos a 90°
- ✓ Pies apoyados en el suelo
- ✓ Espalda recta

### Al Levantar Cargas
- ✓ Flexionar rodillas
- ✓ Mantener espalda recta
- ✓ Pegar carga al cuerpo
- ✓ No girar mientras levantas

### General
- ✓ Pausas cada hora
- ✓ Cambiar de posición frecuentemente
- ✓ Ejercicios de estiramiento
- ✓ Fortalecer core
